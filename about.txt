Скрипт для автомотической проверки инн из Exel.ods(файл линукса) на сайте и получение результатов с записью в текстовый документ.
Скрипт исчет один файл с расширением .ods в текущей дирректории/папке и туда же сохраняет результат в новый файл .txt .
Проверка проходит по странице сайта : https://fedresurs.ru/search/entity?regionNumber=%D0%92%D1%81%D0%B5&onlyActive=true , с использованием библиотеки selenium.


Скачиваем и устанавливаем для развертывания следующие программы:
- IDE
- Гугл хром - тк через него selenium настроен на проверку/имитацию
- Python3.10 +
- Git
- pip


Этапы развертывания:
- делаем git clone https://github.com/.... этого репозитория
- "python3 -m venv venv" создаем виртуальное окружение
- source venv/bin/activate - активируем venv(virtual enverometr) пример для мака и линукс
- pip install reqirements.txt - установка всех нужных пакетов для работы скрипта
- помещаем в текущую папку нужный файл для обработки
- отладка возникших нюансов))

Сокращения слов при обработки:
- "н/д" - нет данных
- "ненайдено" - информация отсутствует или требует уточнения по ФИО в ручную

Обнаружены необработанные исключения:
- некоторые инн, где возникает сложность с обработкой/неточность он сохраняет с пометкой "ненайдено!" - тут требуется ручная обработка.
Пример: по данному ИНН 773800539104 -> "ненайдено" ->list index out of range
